# 07_数字人定制与样片微调建议

状态：`avatar_asset_recommendation_ready` / `local_only_analysis_not_committed`

## 可进入下一步的素材方向

1. 参考帧优先选 `face_visibility=high_close_face` 或 `medium_face_visible` 且 sharpness 较高的片段。
2. 动作生态优先选 `high_motion` 与 `medium_motion` 片段，但必须人工检查手部、商品、手机、身体遮挡。
3. 声音克隆优先选 `clone_suitability=candidate_need_manual_listening` 的片段，并先确认授权、噪声、多人声、背景音乐。
4. 30 秒 / 60 秒样片应同时覆盖近景口播、自然眨眼、嘴角变化、轻微转头、至少一次手势或道具遮挡。
5. 供应商验收不要只看脸；必须同时看眼神、口型、牙齿、肩颈、手势、停顿、语气和画面遮挡。

## 不适合直接使用的素材类型

- 远景、小脸、侧脸过多、脸部遮挡严重的片段。
- 音量过低、背景噪声重、多人重叠说话、直播音乐盖过人声的片段。
- 商品或手机长期挡脸、动作太快导致模糊的片段。
- 只有模型说“看起来可以”的片段；仍需人工复核后才能进入训练/供应商投喂。
