# 参考动作逐条拆解

状态：`reference_motion_breakdown_completed`

## source_01：站立绿幕讲解

- 0-3 秒：人物站立，单手近腰/胯，另一只手开始讲解动作；嘴型和表情活跃。
- 3-6 秒：头部轻微转向，身体有左右晃动，手部从腰部向胸前过渡。
- 6-9 秒：手势向下/向前变化，肩颈有节奏，表情从认真到强调。
- 9-12 秒：两手/腰部动作更明显，身体重心变化。
- 12-15 秒：继续讲解，头部和眼神变化明显，手部节奏持续。

动作难点：手势、肩颈、身体重心和表情同时变化，属于高动作强度样本。

## source_02：躺卧桥式讲解

- 0-3 秒：躺卧在床上，膝盖弯曲，臀部/躯干抬起，双手靠近脸部或胸前。
- 3-6 秒：身体维持桥式姿态，一只手向上/向前表达。
- 6-9 秒：手指/手掌持续变化，头部朝上，脸部角度偏斜。
- 9-12 秒：躯干姿态稳定，手势继续配合说话。
- 12-15 秒：身体基本维持，手部和嘴型仍有变化。

动作难点：脸部角度偏斜、躺卧透视强；粗姿态容易保留，细手势和口型检测不稳定。

## source_03：坐姿瑜伽球讲解

- 0-3 秒：坐姿分腿，坐在小球上，两手向前/向下，嘴型打开。
- 3-6 秒：双手下压或外展，肩部与躯干轻微变化。
- 6-9 秒：手势打开，身体重心微调，脸部表情变化。
- 9-12 秒：双手继续外展/回落，眼神和嘴型有明显变化。
- 12-15 秒：手势回到低位，头部和表情继续跟随说话节奏。

动作难点：双手同步动作和肩部节奏是关键；脸部正面较清楚，适合做人审。
